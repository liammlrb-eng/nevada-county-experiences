For developers

Example: Google Calendar subscribe

1. Open Google Calendar
2. Left sidebar → "Other calendars" → "+ Subscribe to calendar"
3. Paste:  https://liammlrb-eng.github.io/nevada-county-experiences/feeds/events.ics
4. Done — events appear in your calendar and refresh automatically.

Example: JSON fetch (Node, Python, anything)

fetch('https://liammlrb-eng.github.io/nevada-county-experiences/feeds/events.json')
  .then(r => r.json())
  .then(data => {
    console.log(data.count, 'events');
    data.events.forEach(ev => console.log(ev.date, ev.title, '@', ev.venue));
  });

How it works

This site is a federated aggregator. It pulls events from a dozen Nevada County publishers (RSS, JSON APIs, scraped HTML — whatever each venue exposes), filters out duplicates and Sierra-side noise, and presents the union as one trip-planning experience. These feeds are the outbound side of that pipeline — they make the aggregated calendar usable in your tools, not just on this site.

If you publish events for a Nevada County venue and want to make sure we pick them up, see the main site for ways to submit. The easiest is to add schema.org/Event JSON-LD to your existing event page — Google, this site, and other aggregators will all ingest it automatically.
